Bulk Upload Users

How To Bulk Upload Users

To bulk upload users, you must first create a CSV file with the following columns:

  • First Name
  • Last Name
  • Email
  • Department
  • Role

Here is an example of what the CSV file would look like:

First Name,Last Name,Email,Department,Role
John,Doe,john.doe@healthtrio.com,IT,user
Jane,Doe,jane.doe@healthtrio.com,Engineering,admin

Upload CSV File

Once you have created your CSV file, you can upload it here to bulk upload users.

Important! Ensure the first row of the CSV has "First Name,Last Name,Email,Department,Role" and that there are no empty lines/end row.
